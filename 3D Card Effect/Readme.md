# 3D Card Effect

An interactive product card with a realistic 3D tilt parallax effect, responding to mouse movement for an immersive browsing experience.

## ✨ Features

- Real-time 3D card rotation following mouse position
- Depth-based pop-out effect on card elements (title, image, sizes, purchase button)
- Smooth reset animation on mouse leave
- Responsive design with CSS custom properties

## 🛠 Technologies

- HTML5
- CSS3 (Flexbox, `perspective`, `transform-style: preserve-3d`, `translateZ`)
- JavaScript (mouse event tracking, dynamic style manipulation)

## 🚀 Usage

Open `index.html` in your browser. Move your mouse over the card to see the 3D effect.
