# Sign In / Sign Up Form

A dual-purpose authentication form with smooth toggle animation between Sign In and Sign Up views, with social login icons and an SVG illustration.

## ✨ Features

- Toggle between Sign In and Sign Up forms with scale-in animation
- Animated form transition using `@keyframes`
- SVG hero illustration
- Social login icons (Facebook, Twitter, Google) on the Sign Up side
- Clean BEM SCSS structure with custom properties
- Responsive design — single column on mobile, two-column on desktop

## 🛠 Technologies

- HTML5
- SCSS (BEM nesting, variables, `@keyframes`)
- JavaScript (toggle logic, class management)
- Boxicons

## 🚀 Usage

Open `index.html` in your browser. Click "Sign Up" or "Sign In" to toggle between forms.
