# Pricing Cards

![Preview](./Preview.png)

A three-tier pricing table with gradient pricing badges, checkmark feature lists, and hover shadow effects.

## ✨ Features

- Three pricing tiers: Basic, Professional, Enterprise
- Gradient pricing badges with CSS pseudo-element arrow effects
- Feature checkmark lists with icons
- Hover shadow depth on each card
- Fully responsive grid layout (1 → 2 → 3 columns)
- HSL-based color system for easy theme customization

## 🛠 Technologies

- HTML5
- SCSS (BEM nesting, HSL custom properties, pseudo-elements)
- Unicons

## 🚀 Usage

Open `index.html` in your browser.
