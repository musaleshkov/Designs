# Login Form

![Preview](./Preview.png)

An elegant login form with floating labels that animate on focus, decorative radial gradient shapes, and social login icons.

## ✨ Features

- Floating label animation — labels slide up and change color on input focus
- Decorative circular gradient shapes in the background
- Icons integrated into input fields
- Social login buttons (Facebook, Google, Instagram)
- Responsive layout switching from single-column to two-column on desktop

## 🛠 Technologies

- HTML5
- CSS3 (Flexbox, custom properties, transitions)
- JavaScript (focus/blur event handling)
- Boxicons

## 🚀 Usage

Open `index.html` in your browser. Click into input fields to see the floating label effect.
