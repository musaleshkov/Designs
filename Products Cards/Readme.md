# Product Shop Cards

A responsive grid of product cards with a unique hover effect — a vertical label slides in from the left, the product image tilts, and pricing shifts.

## ✨ Features

- Vertical-rl rotated product name label that slides in on hover
- Product image rotation on card hover
- Favorite and cart icon buttons
- Original and sale pricing display
- BEM modifier-based color theming per card (blue, cyan, peach)
- Auto-fit responsive grid

## 🛠 Technologies

- HTML5
- CSS3 (Grid, `writing-mode: vertical-rl`, transforms, transitions)
- Ionicons

## 🚀 Usage

Open `index.html` in your browser. Hover over any product card.
