# AirPods Landing Page

A beautifully animated landing page for AirPods featuring GSAP entrance animations and ScrollMagic scroll-driven reveals, written in Spanish.

## ✨ Features

- GSAP 3 staggered entrance animations (text, images, navigation items)
- ScrollMagic-triggered reveal for the specifications section
- Floating/breathing CSS keyframe animations on AirPod images
- Responsive mobile-first design with hamburger menu
- BEM-named SCSS with CSS custom properties

## 🛠 Technologies

- HTML5
- SCSS (nesting, variables, BEM)
- GSAP 3 (GreenSock Animation Platform)
- ScrollMagic (scroll-driven animations)
- Google Fonts (Roboto)

## 🚀 Usage

Open `index.html` in your browser. Scroll down to see the trigger animations.
