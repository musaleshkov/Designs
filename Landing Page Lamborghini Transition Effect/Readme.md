# Lamborghini Landing Page

A dark-themed supercar landing page with GSAP-powered entrance animations and performance statistic cards.

## ✨ Features

- GSAP 3 staggered entrance animations (title, image, description, stats)
- Dark luxury-themed color palette
- Vertical-rl rotated title for dramatic visual impact
- Performance spec cards with shadow depth
- Responsive mobile-first navigation with hamburger menu

## 🛠 Technologies

- HTML5
- SCSS (BEM nesting, variables, grid)
- GSAP 3
- Ionicons

## 🚀 Usage

Open `index.html` in your browser. Watch the animated elements fade and slide in.
