# Landing Slider Page

![Preview](./preview.png)
![Preview 1](./Preview_1.png)

A fullscreen image slider showcasing tropical island destinations with Swiper.js transitions and GSAP text animations.

## ✨ Features

- Fullscreen Swiper.js slider with fade transitions
- Thumbnail navigation gallery at the bottom
- GSAP text animations triggered on slide change
- Embedded YouTube video popup
- Responsive mobile-first layout

## 🛠 Technologies

- HTML5
- SCSS (BEM nesting, grid, custom properties)
- Swiper.js
- GSAP 3
- Boxicons

## 🚀 Usage

Open `index.html` in your browser. Click thumbnails to change slides and see animations.
