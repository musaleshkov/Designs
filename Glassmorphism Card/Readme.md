# Glassmorphism Cards

A stunning frosted-glass (glassmorphism) card layout with circular gradient decorations, backdrop blur, and animated social icons on hover.

## ✨ Features

- Glassmorphism effect using `backdrop-filter: blur()` and semi-transparent gradients
- Decorative radial gradient circles positioned behind cards
- Slide-in social media icons on card hover with staggered transitions
- Layered gradient buttons with depth via inset/outer box-shadows
- Fully responsive grid layout (1 → 2 → 3 columns)

## 🛠 Technologies

- HTML5
- SCSS (BEM nesting, variables)
- Boxicons

## 🚀 Usage

Open `index.html` in your browser. Hover over a card to reveal social links.
