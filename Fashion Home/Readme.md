# Fashion Home Landing Page

A clean, minimalist fashion brand landing page with a hero layout, product imagery, and social media links.

## ✨ Features

- Fullscreen background image with hero text overlay
- Gradient call-to-action button
- Vertical social media link bar with CSS rotation
- Staggered product imagery with absolute positioning

## 🛠 Technologies

- HTML5
- CSS3 (Flexbox, `background-image`, `transform: rotate`, absolute positioning)

## 🚀 Usage

Open `index.html` in your browser.
